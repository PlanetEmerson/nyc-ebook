Posez votre question sur New York

Vous préparez un voyage? Vous voulez un conseil que vous ne trouverez pas dans les guides?

Je lis chaque message et je réponds personnellement.

Le formulaire est idéal pour les questions sur un premier voyage, un quartier précis, ou le contexte du livre. Si vous cherchez d'abord à vous orienter, commencez par ce guide de départ ou par le blog New York.

Je réponds généralement sous 48-72h. Pour les questions urgentes de voyage, précisez vos dates.

Avant d'écrire

  • Si vous préparez un premier séjour, consultez Premier voyage à New York.
  • Si vous voulez sentir la ville autrement, commencez par le livre.
  • Si vous préférez lire avant de poser une question, explorez les articles.
