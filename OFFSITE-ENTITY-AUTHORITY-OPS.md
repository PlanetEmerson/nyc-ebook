# Off-site Entity And Authority Ops

## Canonical Identity
- Use the exact public name `F.B. Emerson` everywhere. Do not alternate with `FB Emerson` or `F. B. Emerson`.
- Use one short bio across profiles: `Auteure française de "New York, Mon Éveil", récit de transformation à travers New York.`
- Use the same primary URL everywhere: `https://fbemerson.com/a-propos/` for author identity, `https://fbemerson.com/new-york/` for book mentions.

## Profiles To Complete
- Goodreads Author: claim the profile, add the exact bio, book cover, author photo, site link, and make `New York, Mon Éveil` the featured title.
- Amazon Author Central: mirror the same name, bio, photo, and link the main site plus the book page.
- LinkedIn: create or update an author-facing profile with the same bio, featured book link, and a pinned post pointing to the book page.
- Instagram and Pinterest: keep the same bio line, link-in-bio destination, and book cover/headshot pairing.

## Search Console And Indexing
- Submit `https://fbemerson.com/sitemap.xml` in Google Search Console and Bing Webmaster Tools.
- Inspect `/new-york/`, `/new-york/premier-voyage/`, `/a-propos/`, and `/new-york/blog/` after major updates.
- Track branded queries for `F.B. Emerson` separately from non-branded New York/book queries.

## Mentions And Link Targets
- Primary link targets are route-specific. Use the page that matches the mention type instead of sending every mention to the homepage.
- `/new-york/` for book reviews, interviews, reading lists, and purchase-intent coverage.
- `/a-propos/` for author bios, podcasts, guest features, and press mentions.
- `/new-york/premier-voyage/` for travel-resource roundups and first-trip planning mentions.
- `/new-york/blog/` only when a roundup wants a broader editorial hub.

## Outreach Priorities
- French travel newsletters, literary newsletters, and expat/New York editorial sites first.
- Interview and guest-essay angles should point to the author page or the book page, not a random article.
- Resource or planning roundups should point to `/new-york/premier-voyage/`.
- Reclaim unlinked mentions monthly by searching for `F.B. Emerson` and `New York, Mon Éveil`.

## Operating Cadence
- Monthly: refresh bios, links, and featured assets across all profiles.
- Monthly: log new mentions, linked mentions, and unlinked mentions in one sheet.
- Quarterly: review whether the sameAs targets on-site still match the profiles that are actively maintained.
